# ansible-ssh-bootstrap
Простой набор Ansible ролей (плейбуков) для автоматической настройки SSH-соединения между управляющей машиной (Ansible controller) и целевыми хостами.

## 📋 Описание

Этот проект предназначен для автоматизации процесса подготовки целевых хостов к управлению через Ansible. Он создает dedicated пользователя, настраивает sudo без пароля и устанавливает SSH-ключи для безопасного подключения.

## 🚀 Возможности

- Создание dedicated пользователя для Ansible
- Настройка sudo без пароля
- Автоматическое добавление SSH публичных ключей
- Настройка правильных прав доступа для .ssh директории
- Поддержка групповых и индивидуальных конфигураций

📝 Ручная настройка (shell скрипт)
Если по каким-то причинам вы не можете использовать Ansible, прилагается простой shell скрипт для ручной настройки:

bash
# Сделайте скрипт исполняемым
chmod +x manual_setup.sh

# Запустите скрипт на целевом хосте (от root)
./manual_setup.sh

По возможности подключите использование ansible-vault для шифрования ключей.
